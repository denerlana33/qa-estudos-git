# Casos de Teste – Login

## CT01 – Login com credenciais válidas
- Dado que o usuário possui credenciais válidas
- Quando informar usuário e senha corretos
- Então o sistema deve permitir o acesso

## CT02 – Login com senha inválida
- Dado que o usuário informe senha incorreta
- Quando tentar acessar
- Então o sistema deve exibir mensagem de erro
